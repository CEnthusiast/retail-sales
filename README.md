# Retail Sales Project

A project about retail sales dataset from [Kaggle](https://www.kaggle.com/datasets/mohammadtalib786/retail-sales-dataset), using Databricks for the cleaning and PowerBI for the visualization

## [Databricks](https://dbc-18c09656-1ce4.cloud.databricks.com/browse/folders/3657252745044140?o=7474648061881257)
- Processes data using Apache Spark
- Transformed data is stored in Delta Lake tables
